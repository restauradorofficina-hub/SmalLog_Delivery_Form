📦 SmalLog — Política de Privacidade

Versão 1.1 · Última atualização: Abril de 2026

ℹ️ O SmalLog não possui servidor próprio. Todos os seus dados ficam armazenados exclusivamente no seu dispositivo. Nenhuma informação é transmitida para servidores externos.

1. Quem somos

O SmalLog é um aplicativo de gestão de entregas para entregadores autônomos, desenvolvido pela AgAiTec. Contato: smallog.app@gmail.com

2. Dados coletados

O aplicativo coleta e armazena localmente:

  • Nome e telefone do usuário (cadastro)
  • Fotos de etiquetas e pacotes (capturadas pelo usuário)
  • Endereços e CEPs de entrega
  • Histórico de entregas realizadas
  • Dados de veículo (tipo, capacidade)

3. O que NÃO coletamos

  • Localização em tempo real
  • Dados bancários ou financeiros
  • CPF, RG ou documentos pessoais
  • Contatos do celular
  • Dados de navegação ou comportamento

4. Como usamos os dados

Os dados são usados exclusivamente para:

  • Gerenciar e roteirizar as entregas do usuário
  • Gerar comprovantes de entrega via WhatsApp
  • Exibir histórico e relatórios locais

5. Compartilhamento

Não compartilhamos, vendemos ou transferimos seus dados para terceiros. O único envio externo é o comprovante de entrega que o próprio usuário envia pelo WhatsApp ao remetente.

6. Serviços externos utilizados

  • ViaCEP — consulta de endereço por CEP (envia apenas o CEP, sem dados pessoais)
  • Nominatim/OpenStreetMap — geocodificação de endereços para roteirização
  • Google Maps / Waze — navegação (abre o app externo)

7. Armazenamento e segurança

Todos os dados são armazenados localmente usando o banco de dados Hive, protegido pelo sistema operacional Android. Não existe backup automático em nuvem. Se o aplicativo for desinstalado, os dados são apagados.

8. Seus direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Acesso: ver todos os dados em Histórico de Entregas
  • Exclusão: usar "Resetar aplicativo" no Perfil para apagar tudo
  • Portabilidade: exportar via relatórios em PDF
  • Revogação: desinstalar o app remove todos os dados

9. Crianças

O SmalLog é destinado a entregadores adultos e profissionais. Não coletamos dados de menores de 18 anos.

10. Alterações nesta política

Eventuais alterações serão comunicadas na tela de consentimento do app no próximo acesso após a atualização.

11. Contato

Dúvidas sobre privacidade: smallog.app@gmail.com
